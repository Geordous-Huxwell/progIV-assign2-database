TOOL TOWN API docs

Item Service

Function Method Route Parameters
List all items GET /api/item none
List item details based on item ID GET /api/item/{id} id: number
Search for items by name GET /api/item/{searchQuery} searchQuery: string
Add new item to inventory POST /api/item body:
 {
  id: number
  name: string
  quantity: number
  price: number
  supplierId: number
  }
Increase or decrease item quantity PUT /api/item body:
 {
 id: number
 quantity: number
 }
Remove item from inventory DELETE api/item/{id} id: number

Supplier Service

Function Method Route Parameters
List all suppliers GET /api/supplier none
List supplier details based on supplier ID GET /api/supplier/{id} id: number
Search for supplier by name GET /api/supplier/{searchQuery} searchQuery: string
Add new supplier to inventory POST /api/supplier body:
 {
  id: number
  name: string
  address: string
  contact: string (optional)
  }
Remove supplier from supplier list DELETE api/supplier/{id} id: number
